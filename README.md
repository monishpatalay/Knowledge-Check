# Knowledge Check — Simon Says Game

A "Simon Says" memory game: press any key to start, watch the color/sound sequence, then repeat it by clicking the matching buttons. Each correct round adds one more step to the sequence; a wrong click resets the game.

**Live site:** https://monishpatalay.github.io/Knowledge-Check/

## Features

- Randomly generated, ever-growing color sequence (red, blue, green, yellow)
- Sound + button flash animation for each step
- Game-over animation and level reset on a wrong click

## Tech Stack

HTML5, CSS3, JavaScript, jQuery.

## Running It

```bash
open index.html
```

Press any key to begin, then click the buttons in the sequence shown.
